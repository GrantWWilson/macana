Detailed Description

TimePlace - keeps track of where and when.

The TimePlace object simply keeps the time and telescope location.

Todo:
At some point we should add the UT correction predictions so that the ASTE ra/dec to az/el (and visa versa) transformations are more accurate.

bool TimePlace::getASTETiming (  )  [protected]

collects ASTE timing data and signal conditions it

Collects the ASTE timing data and then applies the standard signal conditioning. Also applies the time offset which is a fixed latency (at least it's assumed to be fixed!) between the detector signals and the telescope signals. Much of this code mimics what is found in aztec_makeASTEPointing.pro.
