Detailed Description

Map - the base class of imaging products.

A Map is everything associated with an image of a field. Map objects are used in the Observation, Coaddition, and NoiseRealization classes. To save memory I have not included the absolute sky matrices (ra and dec) in this class since they are shared amongst many maps.

The weight map is common amongst many maps and so as-implemented with a direct copy this is very wastful in terms of memory. Figure out how to pass this weight map by reference isntead.

bool Map::fitToGaussian (  ) 

fits image to gaussian located near map center

Fits the Map image to a 2-d gaussian using the gsl nonlinear least squares fitter as implemented in the Utility gaussFit. There are a few things in here that are hard coded that should be generalized. The output best-fit parameters are written into the map's netcdf file as attributes to the map that has been fit.
