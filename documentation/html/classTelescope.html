bool Telescope::getLMTPointing (  ) 

collect and signal condition the LMT pointing signals

The LMT pointing is easy since it's all right there in the data file. Collect it, do some signal conditioning to remove dropouts, NaNs, and spikes, correct the rollovers if needed, and align with the detector signals.
